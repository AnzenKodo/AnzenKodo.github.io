C Complier

Even with these positives there still many negatives with them. For example, let's take clang. It support every architecture imaginable, but with it comes unnecessary which most of the time you don't need to make your program.

Today developers expect development tools will bloat there system. But it don't have to be this way. There are many people spoken up about today's world software bloat, and working on improving the problem that we have created.
