Set Logic

  • ∈\in∈ - is member of.
    • x∈(0,1)x \in (0, 1)x∈(0,1) that means 0<x<10 < x < 10<x<1.
    • x∈[0,1)x \in [0, 1)x∈[0,1) that means 0≤x0 \leq x0≤x.
      • The square bracket ([[[) means ≤\leq≤.
    • x∈[0,1]x \in [0, 1]x∈[0,1] that means 0≤x≤10 \leq x \leq 10≤x≤1.
    • A∈BA \in BA∈B - is the set of all objects that are a member of A, or B, or both.
  • ∉\notin∉ - is not member of
  • ∋\ni∋ - owns (has member).
  • ⊂\subset⊂ - is proper subset of.
  • ⊆\subseteq⊆ - is subset of.
    • If all the members of set A are also members of set B, then A is a subset of B, denoted A ⊆ B.
    • 1,2{1, 2}1,2 is a subset of 1,2,3{1, 2, 3}1,2,3, and so is 2{2}2 but 1,4{1, 4}1,4 is not.
  • ⊃\supset⊃ - is proper superset of.
  • ⊆\subseteq⊆ - is superset of.

Operations

  • ∪\cup∪ - set union.
    • A ∪ B, is the set of all objects that are a member of A, or B, or both.
    • The union of {1, 2, 3} and {2, 3, 4} is the set {1, 2, 3, 4}.
  • ∩\cap∩ - set intersection.
    • A ∩ B, is the set of all objects that are members of both A and B.
    • The intersection of {1, 2, 3} and {2, 3, 4} is the set {2, 3}.
  • ∖\setminus∖ - set difference.
    • U \ A, is the set of all members of U that are not members of A.
    • The set difference {1, 2, 3} \ {2, 3, 4} is {1}, while conversely, the set difference {2, 3, 4} \ {1, 2, 3} is {4}.
