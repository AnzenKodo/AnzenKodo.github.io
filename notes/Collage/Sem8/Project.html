What is Nostr?

Nostr, short for "Notes and Other Stuff Transmitted by Relays," is a protocol designed for decentralized, censorship-resistant social media.

It is not an app or service but a standard, similar to HTTP or TCP-IP, upon which anyone can build.

It allows users to publish and receive messages globally without relying on a central server. Instead, it uses a network of relays‚Äîsimple servers that store and distribute data.

How Does Nostr Differ from Blockchain?

  • Purpose: Nostr is for social interaction, while blockchain is for transaction recording, like in cryptocurrencies.
  • Data Structure: Nostr uses JSON events distributed via relays, while blockchain chains blocks together.
  • Consensus: Blockchain needs network agreement on the ledger state, but Nostr doesn‚Äôt; it relies on signed events and user choice of relays.
  • Data Persistence: Blockchain data is permanent; Nostr data depends on relays, which might not store events indefinitely.
  • Tokenization: Blockchain often involves tokens (e.g., Bitcoin), but Nostr doesn‚Äôt have its own, though it can integrate with payment systems like the Lightning Network.
