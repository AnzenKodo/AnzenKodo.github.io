Recommendation System

A recommendation system is a type of machine learning system that provides personalized recommendations to users based on their past behaviors, preferences, and patterns.

Types:

  • Content-Based: Recommends items to users based on their past preferences and behaviors.
  • Collaborative Filtering: recommendation systems recommend items to users based on the preferences and behaviors of other slimier users.

Data Stream

A data stream is a flow of data which arrives at uncertain intervals of time.

A Data Stream Management System (DSMS) is a software system designed to manage, process, and analyze continuous, high-velocity data streams in real time.
