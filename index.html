Libraries

All libraries in libs folder are heavily modified version of original libraries.

All the libs used in this project:

  • github_flavored_markdown & sanitized_anchor_name by Dmitri Shuralyov
  • chroma by Alec Thomas
  • gofeed & goxpp by mmcdole
  • blackfriday by Russ Ross
  • text & net by The Go Authors

Indirect

  • goquery by PuerkitoBio
  • cascadia by Andy Balholm
  • regexp2 by Doug Clark
  • jsoniter by json-iterator
  • concurrent & reflect2 by Modern Go Programming

Building & Running

  • Download Go Programming Language.
  • Clone the repository.
  • Go to project directory
  • Run go run .
