AK#Dotfiles

My current setup of NixOS, Qtile, NeoVim, kitty, Xonsh shell, and more information about my mobile setup. Get my old setup on legacy branch.

Installation

  • Install NixOS
  • Fork this repo
  • Clone the forked repo:
    git clone https://github.com/AnzenKodo/dotfiles.git ~/.config/dotfiles
  • Enter the following commands:
    export NIXOS_CONFIG="~/.config/dotfiles/nix/configuration.nix"
sudo nixos-rebuild switch
  • Now Enjoy ;)
